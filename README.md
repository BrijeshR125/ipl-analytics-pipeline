# IPL Analytics Pipeline 🏏

An end-to-end data pipeline built to analyze IPL match data from 2008-2020.

## Tech Stack
- Python (pandas) — data ingestion and transformation
- SQLite — relational database storage
- SQL — data analysis and querying
- Tableau — visualization

## Dashboard
🔗 [View Live Dashboard](https://public.tableau.com/app/profile/brijesh.vishwakarma7334/viz/IPLAnalyticsDashboard_17760055302220/Dashboard1)

## Dataset
- 800+ IPL matches across 13 seasons
- 180,000+ ball-by-ball delivery records
- Source: Kaggle

## What This Project Does
1. Loads raw CSV data using Python pandas
2. Cleans and explores the dataset
3. Stores data in a SQLite relational database
4. Runs SQL queries to extract insights

## Key Insights Found
- 🏆 Most successful team: Mumbai Indians
- 🏏 Highest run scorer: Virat Kohli
- 🎯 Top wicket taker: YS Chahal (213 total)
- 🏟️ Most used venue: Eden Gardens / Wankhede
